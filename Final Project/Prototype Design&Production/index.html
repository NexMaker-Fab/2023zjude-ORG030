Modeling of the tectum

  1. Design and sketch the shell;

  2. Draw basic curves to predetermined dimensions.

  3. Continue drawing the curve.

  4. Lofted the curve into a surface;

  5. Continue drawing curves and lofting into surfaces until the top half is basically completed;

  6. Start drawing the bottom half, using the initial curve.

  7. Extrude the selected curve;

  8. Chamfer the bottom;

  9. Cut the bottom and make holes for the bottom surface;

  10. Draw the inner face of the lower half;

  11. Look at the profile and check whether the interior faces intersect;

  12. Chamfer the model;

  13. Get the final model.

  14. Split the model because it’s too large to print at once.

  15. Segmentation results;

  16. Save the segmented models separately and import them into Tuozhu slicing software;

  17. Because the model is large and time-consuming, it is necessary to adjust the printing parameters to the parameter values that take less time and have better results;

  18. The printing process;

  19. After a long wait, the print is finished;

Modeling of button parts

  1. Use Adobe IIIIustrator spline tool, fixed size, create closed spline line and modify the color, establish the timbre corresponding to the button LOGO, corresponding to the finger, pipa, piano, drums.

  2. Import the AI file into C4D, right click the first item in the object list, select “Current state to object” to split the combined model, and classify the child objects, as shown in the figure.

  3. First use the axis in the tool, center the coordinate axis to the object, stretch the model in the z-axis direction to establish the thickness, and adjust the position of each part of the model to form the top of the button.

  4. Create the cylinder, change the dimensions, move to the back of the model rotation direction, as the bottom of the button.

  5. Put all the models you want to print as a whole in the same parent object, select them all and right-click the connection object to get the model.

  6. Select “File - Export -Wavefront OBJ” Select OBJ format default output to export the model.

Cutting the Tectum

  1. According to the results of the previous discussion, we drew the cutting pattern marked with the parameter size;

  2. Extract the outer contour of the cutting plate in the modeling scheme;

  3. Import into Adobe illustrator;

  4. Draw according to the size and pattern given in step 1, and get the pattern file in ai format;

  5. Extract the spline line of the bottom surface in the modeling file and export it to ai format.

  6. Copy the two documents to the computer where the special education 114 laser cutting machine is located using u disk;

  7. Set different layers for different lines - set power and cutting speed;

  8. Download to laser cutting machine;

  9. Place the board and adjust the position of the cutting head;

  10. Click to start cutting, after a moment, the cutting is finished;
