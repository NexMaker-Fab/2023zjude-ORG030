2 Fusion 360 Install

  1. Go to the app store and find the add-on you want to install;
  1. Click to download;
  1. Double-click the downloaded mis file, you can install successfully;
  1. Enter the software add-on module, find the corresponding application,than you can use.

3 Parameter Design Practice

  1. Create a new sketch and resize it;
  1. Using the Mirror Tool, select all the lines except the blue highlighted line as the mirror segment, and use the blue highlighted line as the mirror line;
  1. Click Finish sketch, click rotate button, select peripheral lines as outline, take the bottom line as axis, rotate 360 degrees;
  1. Sketch, select the section shown on the graph, create the sketch and resize it;
  1. Select the horizontal function, select the object axis and the bottom left corner of the sketch to align the object and sketch;
  1. Adjust the distance between them to 228;
  1. Adjust the sketch bottom line to the construction line;
  1. Using the symmetry command we used earlier, symmetry and complete the sketch;
  1. Rotate the sketch using the rotate command, with the vertical axis as the axis;
  1. Using the split entity command, the entity was divided into the shape of the next stage in turn;
  1. Using the automated modeling command, select the blue entity as the face to connect to and the red entity as the face to avoid;
  1. Click Generate shape, and wait about a minute;
  1. Compare the generated scheme, choose the one that meets your needs, and click Confirm;

4 Constructive Process

Trim the excess arcs using the trimming tool;

Move and copy to create duplicates;

Create a new arc and modify its diameter;

Create a complete slot, and adjust the dimensions;

Extrude the sketch;

Continue to extrude the object in the image;

Select the surface as shown in the image to create a sketch;

After constructing auxiliary lines, create a new circle;

Choose path array, select feature as the object type, choose the object and path, direction select path direction;

Select the shape, create a sketch, generate projection;

Create rectangles and array them;

Select the chosen part, and extrude;

Continue to extrude and assign materials;

Select the corresponding position on the timeline, mirror features;

Select the top plane, extrude; select the top plane to create a sketch, as shown in the image create a new circle and extrude;

Select the circle, extrude according to the parameters shown in the image;

After extruding the circle, create circles on the plane, adjust the distance to 52;

After extruding the object, click on circular array, select feature as object type, choose the corresponding timeline, modify the quantity to 8;

In the right view, create a new sketch and modify dimensions;

Select all sketches to extrude, start by selecting objects, then directly select the main body’s right side, modify the distance to 8;

Select the large circle to extrude, set the range type to target object, select the main body side;

Create a circle sketch, and extrude, distance is -10;

Use mirror, select the corresponding feature on the timeline, after selecting the mirror plane, choose the same calculation type;

After creating a new sketch on the side, create circles and adjust the distance;

After extruding the circles, circular array, select feature, modify the quantity to 8;

In the front view create a sketch, open Sketch 1, project and offset the blue inner circle;

Click on the line, and connect the positions on the image in sequence;

Extrude the respective area, and rectangular array, select feature as the object type, choose the object and axis then modify the quantity to 8, distance to 140;

Find spurGear in the add-on module, click to run;

Adjust the corresponding parameters;

Select all entities, right-click - create parts from entities;

5 Test Motion Link

  1. Using the Assembly - Join command, select the gear and the corresponding shaft in turn, and click OK;
  1. Adjust the position of two gears;
  1. Using the assembly-motion link command, select two gears in turn and adjust the Angle to -360deg;
  1. The link of gears can be realized.
